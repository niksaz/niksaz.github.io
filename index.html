Highlights

We won the Lane Following part competition at AIDO-3 at NeurIPS 2019! I implemented an imitation learning algorithm, its training regime on several datasets and its architecture aimed at each neuron's independence.
Trained a Dota 2 agent using replays of high-rated players available online. Check out the code on GitHub. You can also watch sample episodes here and here.
Participated in AIDO-1 and AIDO-2 competitions. We took the 4-th place at the first competition using the end-to-end reinforcement learning algorithm, and won the second competition by integrating a road segmentation algorithm into a conventional autonomous driving stack. The code for AIDO-2 is on GitHub.
Developed a plugin for IntelliJ based IDEs that allows you to lookup relevant to your editor context documentation on Stack Overflow. The plugin is available for download and open-sourced.
Placed at the top 10%, winning a silver medal at the International Olympiad in Informatics (IOI) 2015.
